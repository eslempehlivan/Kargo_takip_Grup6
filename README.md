# Kargo Takip ve Yönetim Sistemi

## Proje Hakkında

Kargo Takip ve Yönetim Sistemi, kargo süreçlerini dijitalleştiren, gönderi oluşturma, takip ve yönetimi kolaylaştıran web tabanlı bir uygulamadır.

Sistem sayesinde kargolar gerçek zamanlı takip edilir, hareket geçmişi kayıt altına alınır ve admin panel üzerinden tüm süreçler yönetilebilir.

## Hedefler

- Gerçek zamanlı kargo takibi
- Kargo durum güncellemeleri (hazırlandı, yolda, teslim edildi vb.)
- Role-Based Access Control (Admin / Kullanıcı)
- Kargo hareket geçmişi
- Hata yönetimi ve log kayıtları
- Basit ve kullanıcı dostu arayüz
