## 🟢 4. Hafta: Tamamlama, Geliştirme ve Test
Bu hafta boyunca admin paneli tamamlanacak, tüm sistem kapsamlı bir şekildetest edilecek ve karşılaşılan hatalar giderilecektir.
Backend tarafında, admin kullanıcılarının kargo ekleme, listeleme, güncelleme ve silme işlemlerini yapabileceği API'ler geliştirilecektir.
Ayrıca tüm endpoint'ler için testler yazılacak ve performans optimizasyonu sağlanacaktır. 
### Teslim Edilecekler

| No | Teslim Edilecekler |
|----|---------------------|
| 1 | Admin panel (kargo ekleme, listeleme) |
| 2 | API testleri |
| 3 | Hata düzeltmeleri |
| 4 | Tüm sistem test edildi |
| 5 | Proje final hali |

---

###  Backend Görevleri

| Görev | Detay |
|-------|-------|
| Kargo ekleme API'si | POST /cargo (yeni kargo oluşturma - sadece admin) |
| Kargo listeleme API'si | GET /cargo (tüm kargoları listeleme - sadece admin) |
| Kargo güncelleme API'si | PUT /cargo/:id (kargo bilgilerini güncelleme - sadece admin) |
| Kargo silme API'si | DELETE /cargo/:id (kargo silme - sadece admin) |
| API testleri | Tüm endpoint'ler için unit test ve integration test |
| Hata düzeltmeleri | Sprint boyunca bulunan hataların giderilmesi |
| Performans optimizasyonu | Veritabanı sorgularının optimizasyonu, indeksleme |
| Log kayıtları | Sistem loglarının düzenli tutulması |


---

###  Frontend Görevleri 

| Görev | Detay |
|-------|-------|
| Admin panel ana sayfası | Tüm kargoların listelendiği dashboard |
| Kargo ekleme formu | Yeni kargo ekleme sayfası (form + validasyon)  |
| Kargo düzenleme formu | Mevcut kargoyu düzenleme sayfası|
| Kargo silme butonu | Kargo silme (onay modalı ile) |
| Arama ve filtreleme | Admin panelinde kargo arama ve filtreleme  |
| Sayfalama (Pagination) | Çok sayıda kargo için sayfalama sistemi |
| Responsive düzeltmeler | Tüm sayfaların mobil uyumluluğu  |
| Hata düzeltmeleri | Sprint boyunca bulunan frontend hatalarının giderilmesi |

### 4. Hafta – Projenin Tamamlanması

4. haftada projemizin son hali tamamlanmıştır. Bu hafta içinde admin paneli geliştirme süreci bitirilmiş ve sistem genel olarak test edilmiştir.
Testler sırasında çıkan hatalar düzeltilmiştir.Backend tarafında admin kullanıcılarının kargo ekleme, listeleme, güncelleme ve silme işlemlerini
yapabilmesi için gerekli API’ler hazırlanmıştır. Ayrıca test edilmiş ve sistemin daha düzgün çalışması için bazı performans düzenlemeleri
yapılmıştır.Frontend tarafında admin dashboard sayfası oluşturulmuş, kargo listeleme tablosu hazırlanmıştır. Sayfaların mobil uyumlu çalışması
sağlanmıştır. Hafta sonunda proje tamamlanarak GitHub’a yüklenmiş ve gerekli dokümantasyonlar eklenmiştir.
